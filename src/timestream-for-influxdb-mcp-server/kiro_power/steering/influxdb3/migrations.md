# Migrations for Timestream for InfluxDB v3

To migrate from Timestream for InfluxDB v2 to v3, see the [Timestream for InfluxDB v2 to v3 migration guide](../influxdb2/migrations.md).

For differences between Timestream for InfluxDB v2 and v3, see [influxdb-2-vs-3.md](../influxdb-2-vs-3.md).
